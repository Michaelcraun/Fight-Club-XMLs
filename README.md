# Fight Club XMLs

A collection of custom XML compendium files for use with the Fight Club family of character management apps by Lion's Den.

This repository contains homebrew and supplemental content organized by game system. The files are intended for import and use alongside the appropriate Fight Club app and ruleset.

## Included Content

### DnD 5e

- `Ancient Races.xml`

### Pathfinder

- `Ancient Races.xml`
- `Bloodrager (Draconic).xml`
- `Gunslinger (Musket Mater).xml`
- `Ninja.xml`
- `Slayer.xml`
- `Warpriest.xml`
- `Witch (Shadow Patron).xml`

## Usage

Use these XML files with the relevant Fight Club application for the matching game system:

- `DnD 5e` content should be used with the compatible 5e Fight Club app setup.
- `Pathfinder` content should be used with the compatible Pathfinder Fight Club app setup.

Import or load the XML files using your normal Fight Club content workflow for homebrew or custom compendium data.

## Notes

- This is an unofficial fan-made content repository.
- Compatibility depends on the app version and the expected XML schema for the target ruleset.
- Review imported content in the app to confirm formatting, feature text, and mechanical behavior.
